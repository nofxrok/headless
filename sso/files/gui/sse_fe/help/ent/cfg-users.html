Manage User Accounts

Each user that manages systems using SaltStack Enterprise Web Console must have the following:

  • An account on the Salt Master with external authentication privileges defined for the commands and systems to which the user has access.
  • A corresponding account in the SaltStack Enterprise Web Console GUI with the same username and password. SaltStack Enterprise Web Console uses these credentials to connect to the Salt Master using the API.

Add the Account on the Master(s)

Complete these steps to add this account on each Master to which the user needs access.
  1. Add the user on the system using the useradd or similar command.
  2. In the /ect/salt/master configuration file, add the External Authentication System settings required for this user. For example:
    external_auth:
  pam:
    thatch:
      - 'web*':
        - test.*
        - network.*
    steve:
      - .*
  3. Restart the Salt Master after adding user accounts or changing privileges.

Add the Account in SaltStack Enterprise Web Console

  1. Click Configure > Users > Add User.
  2. Provide the user account details and then click Add.
    Option Description

    User Name

    		 The username of the user that you added in the previous step.

    Password

    		 The password of the user that you added in the previous step.
