salt.modules.openbsdservice

The service module for OpenBSD

available

Syntax

salt.modules.openbsdservice.available(
								name
								)

Details

New in version 2014.7.0.

Returns True if the specified service is available, otherwise returns False.

CLI Example:

salt '*' service.available sshd

disabled

Syntax

salt.modules.openbsdservice.disabled(
								name
								)

Details

New in version 2014.7.0.

Return True if the named service is disabled, false otherwise

CLI Example:

salt '*' service.disabled <service name>

missing

Syntax

								name
								)

Details

CLI Example:
