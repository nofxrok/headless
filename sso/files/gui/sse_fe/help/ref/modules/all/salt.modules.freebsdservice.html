salt.modules.freebsdservice

The service module for FreeBSD

available

Syntax

salt.modules.freebsdservice.available(
								name
								)

Details

Check that the given service is available.

CLI Example:

salt '*' service.available sshd

disable

Syntax

salt.modules.freebsdservice.disable(
								name,
								**kwargs
								)

Details

Disable the named service to start at boot

Arguments the same as for enable()

CLI Example:

salt '*' service.disable <service name>

disabled

Syntax

salt.modules.freebsdservice.disabled(
								name
								)

Details

Return True if the named service is enabled, false otherwise

CLI Example:

salt '*' service.disabled <service name>

enable

Syntax

salt.modules.freebsdservice.enable(
								name,
								**kwargs
								)

Enable the named service to start at boot

Arguments

Name Description
name

service name
config

Config file for managing service. If config value is empty string, then /etc/rc.conf.d/<service> used. See man rc.conf(5) for details.

Also service.config variable can be used to change default.

Details

CLI Example:

salt '*' service.enable <service name>
