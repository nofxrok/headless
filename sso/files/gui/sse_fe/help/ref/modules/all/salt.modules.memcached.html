salt.modules.memcached

Module for Management of Memcached Keys

New in version 2014.1.0.

add

Syntax

salt.modules.memcached.add(
								key,
								value,
								host='127.0.0.1',
								port=11211,
								time=0,
								min_compress_len=0
								)

Details

Add a key to the memcached server, but only if it does not exist. Returns False if the key already exists.

CLI Example:

salt '*' memcached.add <key> <value>

decrement

Syntax

salt.modules.memcached.decrement(
								key,
								delta=1,
								host='127.0.0.1',
								port=11211
								)

Details

Decrement the value of a key

CLI Example:

salt '*' memcached.decrement <key>
salt '*' memcached.decrement <key> 2

delete

Syntax

salt.modules.memcached.delete(
								key,
								host='127.0.0.1',
								port=11211,
								time=0
								)

Details

Delete a key from memcache server

CLI Example:

salt '*' memcached.delete <key>

get

Syntax

salt.modules.memcached.get(
								key,
								host='127.0.0.1',
								port=11211
								)

Details

Retrieve value for a key

CLI Example:

salt '*' memcached.get <key>

increment

salt.modules.memcached.increment(
								key,

								host='127.0.0.1',
								port=11211
								)

Details

Increment the value of a key

CLI Example:

salt '*' memcached.increment <key>

replace

Syntax

salt.modules.memcached.replace(
								key,
								value,
								host='127.0.0.1',
								port=11211,
								time=0,
								min_compress_len=0
								)

Details

Replace a key on the memcached server. This only succeeds if the key already exists. This is the opposite of memcached.add

CLI Example:

salt '*' memcached.replace <key> <value>

set_

Syntax

salt.modules.memcached.set_(
								key,
								value,
								host='127.0.0.1',
								port=11211,
								time=0,
								min_compress_len=0
								)

Details

Set a key on the memcached server, overwriting the value if it exists.

CLI Example:

salt '*' memcached.set <key> <value>

Syntax
