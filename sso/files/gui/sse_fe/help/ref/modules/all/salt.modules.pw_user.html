salt.modules.pw_user

Manage users with the useradd command

add

Syntax

salt.modules.pw_user.add(
								name,
								uid=None,
								gid=None,
								groups=None,
								home=None,
								shell=None,
								unique=True,
								fullname='',
								roomnumber='',
								workphone='',
								homephone='',
								createhome=True,
								**kwargs
								)

Details

Add a user to the minion

CLI Example:

salt '*' user.add name <uid> <gid> <groups> <home> <shell>
