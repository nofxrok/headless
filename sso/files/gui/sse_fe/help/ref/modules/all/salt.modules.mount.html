salt.modules.mount

Salt module to manage unix mounts and the fstab file

active

Syntax

salt.modules.mount.active(
								extended=False
								)

Details

List the active mounts.

CLI Example:

salt '*' mount.active

automaster

Syntax

								config='/etc/auto_salt'
								)

Details

CLI Example:

salt '*' mount.fstab

								config='/etc/fstab'
								)

Details

List the contents of the fstab

CLI Example:

is_fuse_exec

Syntax

salt.modules.mount.is_fuse_exec(
								cmd
								)

Details

Returns true if the command passed is a fuse mountable application.

CLI Example:

salt '*' mount.is_fuse_exec sshfs

is_mounted

Syntax

salt.modules.mount.is_mounted(
								name
								)

Details

New in version 2014.7.0.

Provide information if the path is mounted

CLI Example:

salt '*' mount.is_mounted /mnt/share

Syntax

								name,

								fstype='',

Details

set_automaster

Syntax

salt.modules.mount.set_automaster(
								name,
								device,
								fstype,
								opts='',
								config='/etc/auto_salt',
								test=False,
								**kwargs
								)

Details

Verify that this mount is represented in the auto_salt, change the mount to match the data passed, or add the mount if it is not present.

CLI Example:
