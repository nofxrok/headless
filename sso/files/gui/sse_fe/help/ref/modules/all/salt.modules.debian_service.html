salt.modules.debian_service

Service support for Debian systems (uses update-rc.d and /sbin/service)

available

Syntax

salt.modules.debian_service.available(
								name
								)

Details

Returns True if the specified service is available, otherwise returns False.

CLI Example:

salt '*' service.available sshd

disable

Syntax

salt.modules.debian_service.disable(
								name,
								**kwargs
								)

Details

Disable the named service to start at boot

CLI Example:

salt '*' service.disable <service name>

disabled

Syntax

salt.modules.debian_service.disabled(
								name
								)

Details

Return True if the named service is enabled, false otherwise
