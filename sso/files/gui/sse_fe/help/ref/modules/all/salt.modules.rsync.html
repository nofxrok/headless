salt.modules.rsync

Wrapper for rsync

New in version 2014.1.0.

This data can also be passed into  pillar. Options passed into opts will overwrite options passed into pillar.

config

Syntax

salt.modules.rsync.config(
								confile='/etc/rsyncd.conf'
								)

Details

Return rsync config

CLI Example:

salt '*' rsync.config

rsync

Syntax

salt.modules.rsync.rsync(
								src,
								dst,
								delete=False,
								force=False,
								update=False,
								passwordfile=None,
								exclude=None,
								excludefrom=None
								)

Details

Rsync files from src to dst

CLI Example:

salt '*' rsync.rsync {src} {dst} {delete=True} {update=True} {passwordfile=/etc/pass.crt} {exclude=xx}
salt '*' rsync.rsync {src} {dst} {delete=True} {excludefrom=/xx.ini}

version

Syntax

salt.modules.rsync.version( )

Details

Return rsync version

CLI Example:

salt '*' rsync.version
