salt.modules.netbsd_sysctl

Module for viewing and modifying sysctl parameters

assign

Syntax

salt.modules.netbsd_sysctl.assign(
								name,
								value
								)

Details

Assign a single sysctl parameter for this minion

CLI Example:

salt '*' sysctl.assign net.inet.icmp.icmplim 50

get

Syntax

salt.modules.netbsd_sysctl.get(
								name
								)

Details

Return a single sysctl parameter for this minion

CLI Example:

salt '*' sysctl.get hw.physmem

persist

Syntax

salt.modules.netbsd_sysctl.persist(
								name,
								value,
								config='/etc/sysctl.conf'
								)

Details

Assign and persist a simple sysctl parameter for this minion

CLI Example:

salt '*' sysctl.persist net.inet.icmp.icmplim 50

show

Syntax

salt.modules.netbsd_sysctl.show(
								config_file=False
								)

Details

Return a list of sysctl parameters for this minion

CLI Example:

salt '*' sysctl.show
