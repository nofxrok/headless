salt.modules.win_firewall

Module for configuring Windows Firewall

add_rule

Syntax

salt.modules.win_firewall.add_rule(
								name,
								localport,
								protocol='tcp',
								action='allow',
								dir='in'
								)

Details

New in version 2015.2.0.

Add a new firewall rule

CLI Example:

salt '*' firewall.add_rule "test" "tcp" "8080"

disable

Syntax

salt.modules.win_firewall.disable(
								profile='allprofiles'
								)

Details

Disable all the firewall profiles

CLI Example:

salt '*' firewall.disable

enable

Syntax

salt.modules.win_firewall.enable(
								profile='allprofiles'
								)

Details

Enable firewall profile :param profile: (default: allprofiles)

New in version 2015.2.0.

CLI Example:

salt '*' firewall.enable

get_config

Syntax

salt.modules.win_firewall.get_config( )

Details

Get the status of all the firewall profiles

CLI Example:

salt '*' firewall.get_config

get_rule

Syntax

salt.modules.win_firewall.get_rule(
								name='all'
								)

Details

New in version 2015.2.0.

Get firewall rule(s) info

CLI Example:

salt '*' firewall.get_rule "MyAppPort"
