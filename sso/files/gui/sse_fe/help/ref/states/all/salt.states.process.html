salt.states.process

Process Management

Ensure a process matching a given pattern is absent.

httpd-absent:
  process.absent:
    - name: apache2

absent

Syntax

salt.states.process.absent(
								name,
								user=None,
								signal=None
								)

Ensures that the named command is not running.

Arguments

Name Description
name

The pattern to match.
user

The user process belongs
signal

Signal to send to the process(es).
